## 问题诊断

### 网络问题
**现象**: 实时行情接口超时/拒绝连接
**原因**: 代理设置或网络防火墙限制
**解决**: 
1. 已清除代理环境变量
2. 使用备用数据源（个股历史数据接口正常）
3. 指数成分股接口正常

### 建议的测试流程

由于实时行情受限，建议按以下流程测试：
