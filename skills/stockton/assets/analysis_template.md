# 股票分析报告模板

格式化股票分析输出供 LLM 阅读时使用的模板。

## 个股分析报告

```markdown
# {股票名称} ({股票代码}) 分析报告

## 执行摘要
**信号:** {买入/持有/卖出} | **评分:** {评分}/100 | **趋势:** {看涨/看跌/中性}

## 价格表现
- **当前价格:** ¥{价格:.2f}
- **涨跌:** {涨跌幅:+.2f}%
- **20日收益:** {20日动量:+.2f}%
- **60日收益:** {60日动量:+.2f}%

## 技术分析
### 移动平均线
- **MA5:** ¥{ma5:.2f} ({高于/低于} 价格)
- **MA20:** ¥{ma20:.2f} ({支撑/阻力})
- **MA60:** ¥{ma60:.2f} (趋势: {向上/向下})

### 趋势信号
- **排列:** {多头排列/空头排列/无明显趋势}
- **量比:** {量比:.2f}x ({正常/偏高/高})

## 基本面分析
### 盈利能力 (评分: {评分}/25)
- **ROE:** {roe:.1f}% ({优秀/良好/一般})
- **毛利率:** {毛利率:.1f}%
- **净利率:** {净利率:.1f}%

### 成长性 (评分: {评分}/25)
- **营收增长:** {营收增长:+.1f}%
- **利润增长:** {利润增长:+.1f}%

### 财务安全 (评分: {评分}/20)
- **负债率:** {负债率:.1f}% ({安全/中等/高})

### 估值 (评分: {评分}/20)
- **PE:** {pe:.1f}x ({低估/合理/高估})
- **PB:** {pb:.1f}x

## 综合健康评分: {评分}/100
```

## 市场概览模板

```markdown
# A股市场概览

## 主要指数
| 指数 | 价格 | 涨跌 | 
|------|------|------|
| 上证指数 | {上证价格} | {上证涨跌:+.2f}% |
| 深证成指 | {深证价格} | {深证涨跌:+.2f}% |
| 创业板指 | {创业板价格} | {创业板涨跌:+.2f}% |

## 市场广度
- **上涨:** {上涨家数} 只
- **下跌:** {下跌家数} 只
- **涨停:** {涨停数}
- **跌停:** {跌停数}

## 期货贴水
| 期货 | 贴水 | 年化 | 
|------|------|------|
| IF (沪深300) | {if贴水:+.2f}% | {if年化:+.2f}% |
| IC (中证500) | {ic贴水:+.2f}% | {ic年化:+.2f}% |
```

## 使用说明

### 格式化指南

1. **始终包含:** 日期、数据源
2. **数字取整:** 价格保留2位小数，百分比保留1位
3. **颜色编码:** 使用表情符号快速视觉识别
   - 🟢 正面/好
   - 🟡 中性/中等
   - 🔴 负面/差

### 免责声明示例

```markdown
---
**免责声明:** 本分析仅供信息参考。
**数据源:** {Efinance/Akshare/缓存}  
**最后更新:** {时间戳}
```
