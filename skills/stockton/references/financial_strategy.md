# 财务分析与选股策略指南

本文档详细介绍 Stockton 技能的财务分析模块和智能选股功能的使用方法与策略。

## 财务分析模块

### 核心财务指标解读

#### 1. 盈利能力指标

| 指标 | 计算公式 | 优秀标准 | 分析要点 |
|------|---------|---------|---------|
| **ROE** | 净利润 / 净资产 | >15% | 巴菲特最看重的指标，反映资本利用效率 |
| **ROA** | 净利润 / 总资产 | >8% | 反映资产整体盈利能力 |
| **毛利率** | (营收-成本) / 营收 | >30% | 体现产品定价权和竞争优势 |
| **净利率** | 净利润 / 营收 | >10% | 扣除费用后的真实盈利能力 |
| **营业利润率** | 营业利润 / 营收 | >15% | 核心业务的盈利能力 |

**ROE杜邦分析**:
```
ROE = 净利率 × 总资产周转率 × 权益乘数
```

- **高净利率型**: 如茅台(90%+毛利率)，品牌护城河
- **高周转率型**: 如零售行业，薄利多销
- **高杠杆型**: 如银行业，风险较高

#### 2. 成长能力指标

| 指标 | 优秀标准 | 分析要点 |
|------|---------|---------|
| **营收增长率** | >20% | 高成长公司的门槛 |
| **净利润增长率** | >20% | 利润增长比营收增长更重要 |
| **净资产增长率** | >15% | 反映内生增长能力 |

**成长质量判断**:
- 利润增长 > 营收增长：规模效应显现，盈利能力增强 ✅
- 利润增长 < 营收增长：可能价格战或成本上升 ⚠️
- 现金流增长 > 利润增长：利润有现金支撑 ✅
- 现金流增长 < 利润增长：可能有应收账款风险 ⚠️

#### 3. 偿债能力指标

| 指标 | 计算公式 | 安全标准 | 风险提示 |
|------|---------|---------|---------|
| **资产负债率** | 负债 / 资产 | <50% | >70%财务风险高 |
| **流动比率** | 流动资产 / 流动负债 | >2 | <1可能短期偿债困难 |
| **速动比率** | (流动资产-存货) / 流动负债 | >1 | 剔除存货后的真实偿债能力 |

**不同行业标准**:
- **银行业**: 负债率90%+属正常
- **房地产行业**: 负债率70-80%常见
- **科技行业**: 应保持<50%，轻资产运营

#### 4. 运营效率指标

| 指标 | 计算公式 | 优秀标准 | 分析要点 |
|------|---------|---------|---------|
| **总资产周转率** | 营收 / 总资产 | >0.8 | 资产利用效率 |
| **存货周转率** | 营业成本 / 存货 | 行业差异大 | 越高说明库存管理越好 |
| **应收账款周转率** | 营收 / 应收账款 | >6 | 回款速度，越高越好 |
| **现金转换周期** | 存货周转天数 + 应收周转天数 - 应付周转天数 | <30天 | 越短越好 |

#### 5. 现金流指标

| 指标 | 分析要点 |
|------|---------|
| **经营现金流** | 必须为正，且与净利润匹配 |
| **自由现金流** | 经营现金流 - 资本支出，衡量真实盈利能力 |
| **现金流/净利润** | >1说明利润有现金支撑 |

**现金流质量判断**:
```
优质企业: 经营现金流 > 净利润 > 0
一般企业: 经营现金流 > 0，但 < 净利润
风险企业: 经营现金流 < 0，靠融资维持
```

#### 6. 估值指标

| 指标 | 计算公式 | 低估 | 合理 | 高估 |
|------|---------|------|------|------|
| **PE** | 股价 / 每股收益 | <15 | 15-30 | >40 |
| **PB** | 股价 / 每股净资产 | <1.5 | 1.5-3 | >5 |
| **PS** | 市值 / 营收 | <2 | 2-5 | >10 |
| **股息率** | 分红 / 股价 | >3% | 1-3% | <1% |

**不同行业的合理PE**:
- **银行业**: PE 5-10倍正常
- **消费行业**: PE 20-40倍合理
- **科技行业**: PE 30-60倍可接受
- **新能源/医药**: 可能PE 50-100倍

## 智能选股策略

### 预设策略详解

#### 1. 价值投资 (Value)

**策略理念**: 寻找被市场低估的优质企业

**筛选条件**:
- PE < 20
- PB < 2
- 股息率 > 2%
- ROE > 10%
- 资产负债率 < 60%

**适用投资者**: 稳健型、追求安全边际

**代表股票类型**: 银行、保险、公用事业、传统制造业龙头

**注意事项**:
- 低PE可能是价值陷阱（业绩持续下滑）
- 需确认公司没有结构性问题
- 适合长期持有，不适合短期炒作

#### 2. 成长投资 (Growth)

**策略理念**: 寻找高成长性的公司，享受业绩增长带来的股价提升

**筛选条件**:
- 营收增长 > 20%
- 利润增长 > 20%
- ROE > 15%
- 资产负债率 < 50%

**适用投资者**: 进取型、能承受较高波动

**代表股票类型**: 新能源、半导体、生物医药、高端制造

**注意事项**:
- 高增长可能已充分反映在股价中
- 需关注增长的可持续性
- 估值容忍度可以适当放宽

#### 3. 质量投资 (Quality)

**策略理念**: 寻找盈利质量高、财务稳健的企业

**筛选条件**:
- ROE > 15%
- 资产负债率 < 40%
- 毛利率 > 30%
- 流动比率 > 1.5

**适用投资者**: 注重企业质量的投资者

**代表股票类型**: 消费龙头、医药龙头、科技龙头

**注意事项**:
- 高质量公司通常估值较高
- 适合长期持有
- 波动可能小于成长股

#### 4. 蓝筹股 (Blue Chip)

**策略理念**: 投资大盘蓝筹，享受稳定分红和稳健增长

**筛选条件**:
- 市值 > 500亿
- PE < 25
- 股息率 > 2%
- ROE > 12%

**适用投资者**: 保守型、追求稳定收益

**代表股票类型**: 银行、保险、大型国企、消费巨头

**注意事项**:
- 增长潜力可能有限
- 适合作为投资组合的压舱石
- 分红再投资可产生复利效应

#### 5. 小盘成长 (Small Cap Growth)

**策略理念**: 投资小市值高成长股，博取高收益

**筛选条件**:
- 市值 < 200亿
- 营收增长 > 30%
- 利润增长 > 30%
- ROE > 10%

**适用投资者**: 高风险偏好、追求超额收益

**代表股票类型**: 次新股、细分领域龙头、创新型企业

**注意事项**:
- 波动极大，需严格控制仓位
- 流动性可能较差
- 研究难度高，需深入了解公司

## 综合分析框架

### 选股检查清单

```
□ 基本面检查
  □ 财务评分 > 60
  □ ROE > 10%（成长股>15%）
  □ 营收和利润正增长
  □ 经营现金流为正
  □ 负债率合理

□ 估值检查
  □ PE在合理区间（参考行业）
  □ PB不过高（<5）
  □ 与历史估值比较

□ 技术面检查
  □ 趋势向上或回温支撑
  □ 乖离率 < 5%
  □ 量能配合

□ 市场环境
  □ 大盘非空头排列
  □ 所属板块非冷门
  □ 无重大利空消息

□ 风险控制
  □ 单一股票仓位 < 20%
  □ 设置止损位
  □ 分散行业配置
```

## 常见错误与避免

### 错误1: 只看PE选股

**错误**: 只选PE最低的股票
**后果**: 可能选中价值陷阱（业绩持续下滑的公司）
**纠正**: 结合成长性，使用PEG指标（PE/增长率）

### 错误2: 追高成长股

**错误**: 看到高增长就买入，不管估值
**后果**: 估值回归时亏损严重
**纠正**: 设置PE上限，或等待回调买入

### 错误3: 忽视现金流

**错误**: 只看利润增长，不看现金流
**后果**: 利润可能是账面数字，没有现金支撑
**纠正**: 必须检查经营现金流 > 净利润

### 错误4: 过度集中

**错误**: 把所有资金买一只"最好"的股票
**后果**: 黑天鹅事件导致重大损失
**纠正**: 单只股票仓位不超过20%，分散5-10只

### 错误5: 不止损

**错误**: 亏损后不愿卖出，期待回本
**后果**: 小亏变大亏
**纠正**: 设定止损位（如-8%），严格执行

## 进阶技巧

### 1. 财务数据对比分析

```python
# 对比同行业的财务指标
# 如：比较茅台、五粮液、泸州老窖的ROE和毛利率

stocks = ['600519', '000858', '000568']
for code in stocks:
    fin = analyze_financial(code)
    print(f"{code}: ROE={fin.roe}%, 毛利率={fin.gross_margin}%")
```

### 2. 趋势与财务结合

```python
# 寻找"基本面好+技术面佳"的共振点

# 筛选财务优秀的股票
financial_screen = [
    {'code': '600519', 'financial_score': 85},
    {'code': '000858', 'financial_score': 78},
    # ...
]

# 从中找出技术趋势向上的
tech_bullish = []
for stock in financial_screen:
    tech = analyze_for_llm(stock['code'])
    if tech['trend'] in ['bullish_arrangement', 'strong_bullish']:
        tech_bullish.append(stock)

# 这是最好的买入候选
```

### 3. 季度业绩跟踪

```python
# 定期跟踪持仓股票的季度业绩
# 业绩不及预期时减仓，超预期时加仓

holdings = ['600519', '300750', '000001']
for code in holdings:
    fin = analyze_financial(code)
    
    # 检查最新季度业绩
    if fin.revenue_growth < 10 or fin.profit_growth < 10:
        print(f"{code}: 业绩增长放缓，考虑减仓")
    elif fin.revenue_growth > 30 and fin.profit_growth > 30:
        print(f"{code}: 业绩超预期，可考虑加仓")
```
