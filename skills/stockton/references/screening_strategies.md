# 选股策略参考

所有预设筛选策略的详细说明。

## 策略分类

### 1. 价值策略

重点：低估值 + 强基本面

#### `value`（经典价值）

**标准：**
- PE < 20
- PB < 2
- 股息率 > 2%
- ROE > 10%
- 负债率 < 60%

**逻辑：**
本杰明·格雷厄姆式价值投资。寻找低于内在价值交易且有足够质量过滤的公司。

**适用场景：**
- 熊市或市场回调
- 经济周期末期
- 风险规避环境
- 构建防御性组合

**预期结果：** 全市场20-50只，沪深300中5-15只

### 2. 成长策略

重点：高成长公司

#### `growth`（高成长）

**标准：**
- 营收增长 > 20%
- 利润增长 > 20%
- ROE > 15%
- 负债率 < 50%

**逻辑：**
关注营收和利润增长强劲的公司。质量过滤（ROE、负债）避免投机性成长。

**适用场景：**
- 经济周期早期-中期
- 牛市
- 风险偏好环境
- 构建进取型组合

#### `small_cap_growth`（小盘成长）

**标准：**
- 市值 < 200亿人民币
- 营收增长 > 30%
- 利润增长 > 30%
- ROE > 10%

**逻辑：**
小盘股成长潜力更高但风险也更大。严格的成长标准补偿较低的流动性。

**适用场景：**
- 强牛市
- 流动性充裕环境
- 高风险承受能力

**风险提示：** 小盘股可能流动性差、波动大

### 3. 质量策略

重点：高质量企业

#### `quality`（质量因子）

**标准：**
- ROE > 15%
- 负债率 < 40%
- 毛利率 > 30%
- 流动比率 > 1.5

**逻辑：**
巴菲特式质量投资。关注具有持久竞争优势（高毛利率）和强健资产负债表的公司。

**适用场景：**
- 核心组合构建
- 任何市场条件
- 长期持有

### 4. 蓝筹策略

重点：大盘分红股

#### `blue_chip`（蓝筹分红）

**标准：**
- 市值 > 500亿人民币
- PE < 25
- PB < 3
- ROE > 12%
- 股息率 > 2%

**逻辑：**
持续分红的大盘股提供稳定性和收入。质量过滤确保可持续性。

**适用场景：**
- 保守型组合
- 收入导向策略
- 市场不确定性

### 5. 动量策略

重点：趋势跟踪

#### `momentum`（价格动量）

**标准：**
- 20日收益率 > 10%
- 60日收益率 > 15%
- 价格在MA20之上
- 量比 > 1.0

**评分：**
- 20日动量：+10分（强）、+7（好）、+5（中等）
- 60日动量：+10分（强）、+7（好）、+5（中等）
- 趋势一致性：+5分

**逻辑：**
基于"动量效应"的趋势跟踪策略——表现好的股票在短期内往往继续表现好。

**适用场景：**
- 强趋势市场
- 突破确认
- 短期交易

**风险提示：** 动量可能快速反转；使用止损

#### `dual_momentum`（双动量：绝对+相对）

**标准：**
- 20日收益率 > 5%（绝对动量）
- 60日收益率 > 10%（绝对动量）
- 两期均为正（趋势确认）
- ROE > 8%（质量过滤）
- 负债率 < 60%（安全过滤）

**评分：**
与动量策略相同，加质量奖励。

**逻辑：**
Gary Antonacci 的双动量概念：
1. **绝对动量：** 只投资具有正趋势的资产
2. **相对动量：** 选择最强表现者
3. **质量过滤：** 避免动量陷阱（低质量垃圾股反弹）

**适用场景：**
- 最适合战术配置
- 可用于市场择时（无符合条件时持币）
- 降低回撤 vs 纯动量

**预期行为：**
- 强牛市：多个符合条件，高回报
- 弱/横盘市：很少/无符合条件，保存资本
- 熊市：零符合条件（被绝对动量过滤）

## 多因子评分

所有策略使用多因子评分（0-120分）：

### 财务评分（0-70分）

**盈利能力（0-25）：**
- ROE > 20%：+10
- ROE 15-20%：+8
- ROE 10-15%：+6
- ROE 8-10%：+4

**成长（0-25）：**
- 营收增长 > 30%：+10
- 营收增长 20-30%：+8
- 利润增长 > 30%：+10
- 利润增长 20-30%：+8

**安全（0-20）：**
- 负债率 < 40%：+10
- 负债率 40-50%：+6
- 流动比率 > 2：+10
- 流动比率 1.5-2：+6

### 技术评分（0-50分）

**动量（0-30）：**
- 20日收益 > 20%：+10
- 20日收益 10-20%：+7
- 60日收益 > 30%：+10
- 60日收益 15-30%：+7
- 120日收益 > 50%：+5
- 趋势一致性：+5

**技术形态（0-20）：**
- 多头排列（MA5>MA10>MA20）：+15
- 价格 > MA20 5%：+10
- 量比 > 2：+10

## 策略组合

### 组合构建示例

**保守型（60%价值+40%质量）：**
```python
value_stocks = screen_stocks(strategy='value', top_n=15)
quality_stocks = screen_stocks(strategy='quality', top_n=10)
```

**平衡型（40%价值+30%成长+30%质量）：**
```python
value = screen_stocks(strategy='value', top_n=10)
growth = screen_stocks(strategy='growth', top_n=8)
quality = screen_stocks(strategy='quality', top_n=8)
```

**进取型（50%成长+30%动量+20%小盘）：**
```python
growth = screen_stocks(strategy='growth', top_n=10)
momentum = screen_stocks(strategy='momentum', top_n=6)
small_cap = screen_stocks(strategy='small_cap_growth', top_n=4)
```

## 自定义筛选

创建自定义标准：

```python
from skills.stockton.scripts.stock_screener import ScreenCriteria

criteria = ScreenCriteria(
    pe_max=15,              # 深度价值
    pb_max=1.5,
    roe_min=12,
    revenue_growth_min=10,  # 有一定成长
    market_cap_min=100,     # 中盘+
    momentum_60d_min=10     # 有动量
)

results = screen_by_criteria(criteria, top_n=20)
```

## 指数内筛选

在特定指数内筛选以获得更高质量：

```python
# 大盘价值
screen_stocks_advanced(strategy='value', index_name='沪深300', top_n=10)

# 中盘成长
screen_stocks_advanced(strategy='growth', index_name='中证500', top_n=10)

# 小盘动量
screen_stocks_advanced(strategy='momentum', index_name='中证1000', top_n=10)
```

## 回测考虑

**调仓频率：**
- 动量策略：每周（动量衰减快）
- 成长策略：每月
- 价值策略：每季度（价值需要时间）

**交易成本：**
- 小盘策略：因流动性低，成本更高
- 动量策略：换手率高 = 成本高

**市场状态依赖：**
- 动量最适合趋势市场
- 价值最适合复苏阶段
- 质量在任何市场都有效，但在下跌时跑赢
