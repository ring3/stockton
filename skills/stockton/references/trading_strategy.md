# 趋势交易策略详解

本文档详细介绍 Stockton 技能采用的趋势交易理念和策略。

## 核心概念

### 1. 不追策略（不追高）

**原则**: 宁可错过，不可做错。追求每笔交易的成功率。

**乖离率标准**:
```
乖离率 = (现价 - MA5) / MA5 × 100%
```

| 乖离率范围 | 操作建议 | 风险等级 |
|-----------|---------|---------|
| < -5% | 可能破位，观察 | 🔴 高 |
| -5% ~ -3% | 回温较深，观察支撑 | 🟡 中 |
| -3% ~ 0% | 最佳买点区间 | 🟢 优 |
| 0% ~ 2% | 可小仓介入 | 🟢 优 |
| 2% ~ 5% | 谨慎追高 | 🟡 中 |
| > 5% | 严禁追高 | 🔴 高 |

**关键规则**:
- 乖离率 > 5%：直接判定为"观望"，不买入
- 乖离率 < 2%：最佳买点区间
- 价格回温 MA5 获得支撑：最佳介入时机

### 2. 趋势交易（顺势而为）

**多头排列定义**:
```
MA5 > MA10 > MA20
```

**趋势状态分类**:

| 趋势状态 | 均线排列 | 趋势强度 | 操作建议 |
|---------|---------|---------|---------|
| 强势多头 | MA5>MA10>MA20，间距扩大 | 90 | 积极做多 |
| 多头排列 | MA5>MA10>MA20 | 75 | 顺势做多 |
| 弱势多头 | MA5>MA10, MA10<MA20 | 55 | 谨慎做多 |
| 盘整 | 均线缠绕 | 50 | 观望 |
| 弱势空头 | MA5<MA10, MA10>MA20 | 40 | 减磅 |
| 空头排列 | MA5<MA10<MA20 | 25 | 空仓 |
| 强势空头 | MA5<MA10<MA20，间距扩大 | 10 | 坚决空仓 |

**趋势判断逻辑**:
1. 检查 MA5、MA10、MA20 的大小关系
2. 比较当前均线间距与1日前间距（判断趋势是否加强）
3. 趋势强度评分：0-100分

### 3. 买点偏好（回踩支撑）

**最佳买点**:
- 缩量回温 MA5 获得支撑
- 乖离率在 -3% ~ 0% 之间
- 多头排列确认

**次优买点**:
- 回温 MA10 获得支撑
- 乖离率在 -5% ~ -3% 之间
- MA5 仍在 MA10 之上

**观望情况**:
- 跌破 MA20
- 乖离率 > 5%
- 空头排列

**支撑判断标准**:
```python
MA_SUPPORT_TOLERANCE = 0.02  # 2% 容忍度

# MA5 支撑有效条件
abs(price - MA5) / MA5 <= 0.02 and price >= MA5

# MA10 支撑有效条件
abs(price - MA10) / MA10 <= 0.02 and price >= MA10
```

### 4. 量能分析

**量比计算**:
```
量比 = 当日成交量 / 5日平均成交量
```

**量能状态分类**:

| 量能状态 | 量比范围 | 价格变化 | 含义 | 评分 |
|---------|---------|---------|------|------|
| 缩量回调 | < 0.7 | 下跌 | 洗盘特征，好 | 20分 |
| 放量上涨 | > 1.5 | 上涨 | 多头强劲 | 15分 |
| 量能正常 | 0.7-1.5 | - | 正常波动 | 12分 |
| 缩量上涨 | < 0.7 | 上涨 | 动能不足 | 8分 |
| 放量下跌 | > 1.5 | 下跌 | 出货信号 | 0分 |

**量能偏好排序**:
1. 缩量回调（主力洗盘，抛压减轻）
2. 放量上涨（多头强劲）
3. 量能正常
4. 缩量上涨（上攻动能不足）
5. 放量下跌（出货信号，危险）

## 买入信号生成

### 信号等级

| 综合评分 | 买入信号 | 条件 |

| < 35 | 卖出 | 趋势转弱 |
| - | 强烈卖出 | 空头排列 |

### 买入条件检查清单

```
✅ 多头排列：MA5 > MA10 > MA20
✅ 乖离率 < 5%（不追高）
✅ 量能配合（缩量回调或放量上涨）
⚠️ 无重大利空（需人工确认）
✅ 筹码健康（如有筹码分布数据）
```

## 风险管理

### 止损策略

| 止损类型 | 触发条件 | 操作建议 |
|---------|---------|---------|
| 技术止损 | 跌破 MA20 | 减仓或清仓 |
| 幅度止损 | 亏损 > 8% | 严格止损 |
| 时间止损 | 持有 5 日无起色 | 考虑换股 |

### 仓位管理

| 市场环境 | 建议仓位 |
|---------|---------|
| 强势多头 | 70-100% |
| 多头排列 | 50-70% |
| 弱势多头 | 30-50% |
| 盘整 | 30% 以下 |
| 空头排列 | 空仓或 < 10% |

### 风险排查清单

```
❌ 减持公告（股东、高管减持）
❌ 业绩预减/大幅下滑
❌ 监管处罚/立案调查
❌ 行业政策利空
❌ 大额解禁
```

## 策略优化建议

1. **结合大盘环境**: 大盘强势时提高仓位，大盘弱势时降低仓位
2. **板块轮动**: 关注热点板块，避免冷门股
3. **基本面过滤**: 排除业绩差、有退市风险的股票
4. **消息面监控**: 关注重大利好/利空消息
5. **回测验证**: 对策略进行历史数据回测

## 注意事项

1. **机械信号仅供参考**: 技术分析有滞后性，需结合实际情况
2. **严格执行止损**: 不要让小亏变大亏
3. **控制仓位**: 不要满结单只股票
4. **保持耐心**: 等待最佳买点，不要频繁交易
5. **持续学习**: 市场变化，策略也需要调整
